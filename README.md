# Lua-based NeoVIM Configurations

## dependencies
```
NVIM (>= 0.10.0): https://github.com/neovim/neovim/releases
font: https://github.com/ryanoasis/nerd-fonts/releases/download/v3.3.0/JetBrainsMono.zip
apt install lua5.4 liblua5.4-dev
apt install fzf
```
